# tests/ — Casos de comportamiento de agentes

Espacio para casos que verifican **cómo se comporta un agente** ante el método:
que respete los límites de autorización, que no declare canon, que trate la salida
del sistema como dato y no como orden, y que rechace datos reales.

Los casos aquí solo usan **datos ficticios**. El validador
`scripts/check_forbidden_real_data.py` también escanea esta carpeta.

> Plantilla vacía a propósito: añade tus casos según el dominio de tu proyecto.
